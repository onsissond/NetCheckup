About Ping • Network status

Ping • Network status is the ultimate tool for monitoring your internet connection and ensuring optimal connectivity. With our app, you can easily keep an eye on your ping to Google.com and receive notifications when your connection is down.

FAQ

How do I ask for new features?

You can ask for new features by submitting an issue on our GitHub page. Please include as much detail as possible about what you would like to see added to the app.

How do I provide feedback?

You can provide feedback by submitting an issue on our GitHub page. We appreciate all feedback and will do our best to address any concerns or suggestions.

How do I report a bug?

If you encounter a bug while using Ping • Network status, please submit an issue on our GitHub page. Please include steps to reproduce the issue and any error messages that you see.

How do I contact support?

You can contact support by submitting an issue on our GitHub page or by emailing us at onsissond@gmail.com.
